# Final-Year-Project
The Lost and Found Portal for University is a web-based application designed to help students and university staff report and recover lost items within the campus. The system provides a centralized platform where users can post lost or found items, search for missing belongings, and contact the person who reported them.
This project aims to solve the common problem of lost items in universities by creating an organized and easy-to-use digital platform that improves communication and increases the chances of recovering lost belongings.
